NGIAB

🎉Differentiable HBV 2.0 (δHBV 2.0) and SUMMA integrated in NGIAB

δHBV 2.0 is a differentiable Hydrologiska Byråns Vattenbalansavdelning (HBV) model, using an LSTM and multilayer perceptron (MLP) to learn parameters. Both the hourly and daily variants are now integrated in the new release of NGIAB. More details on the δHBV work can be found in this blog post: Hourly Differentiable Modeling Arrives in the NGIAB-NRDS NextGen Ecosystem.

In addition, this release also includes SUMMA integration to enhance modeling capabilities. Background on the SUMMA integration effort is available here: Moving Hydrologic Prediction Forward — A software integration meeting at the Alabama Water Institute.
